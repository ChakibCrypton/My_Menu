Voici un repository Java, sur un projet Menu qui consiste à servir différents menus avec des accompagments et des boissons à notre utilisateur.
Pour cela le projet comporte 3 classes:
1. Order : Elle contient les méthodes qui seront utiles pour notre projet comme runMenu, celle-ci sont expliqué dans la JavaDoc de la classe.
2. OrderTest : Elle contient des méthodes qui vont tester ceux de la classe order selon le principe TDD en dévekoppement.
3. Main : Elle exécute les méthodes de la classe Order et initialise un objet de cette classe.
